Noteworthy

Despite using the same API as I used in 235, this is definitely not a copy/paste. Firstly, I designed the entire webpage around the use of web components. I used them for the header, amiibo cards, and the footer. This means if I need to make a change to any of these sections, I only have to do so once instead of edit the HTML/CSS for every page.

Furthurmore, by utilizing Bulma, I was able to make a much more clean and professional looking website. I think the best results come from the layout. I tried to replicate Amazon's layout of having search queries on the side rather than the top, and to much success with Bulma.

I also have a handful of small touches in here that will hopefully boost the quality. Mainly, the series query on the app page gets its options from the API, so if a new series is ever added, the program will account for it. I also have the home page randomly load an image from a pool that can be changed by clicking on it.
